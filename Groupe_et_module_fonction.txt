MODULE FONCTION
________________________________________________________________________

Un module fonction est contenu dans un "groupe fonction"
Le groupe fonction peut contenir plusieurs module fonction

------------------------------------------------------------------------
TRANSACTIONS ==> SE37 ou SE80 (à préférer)
------------------------------------------------------------------------

SE80 :
Créer le groupe fonction
	-> double clic sur groupe fonction pour afficher le programme principal

Créer un module fonction dans groupe fonction
	-> clic droit sur groupe fonction / Créer / module fonction

------------------------------------------------------------------------
PARAMETRE D'INTERFACE
------------------------------------------------------------------------

- IMPORTATION	=> USING
- EXPORTATION
- MODIFICATION	=> CHANGING
